### 介绍
> &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;本项目的灵感来源于在外包工作时的绝望与无助,账号`辣么多`,线上的App`辣么多`,官方的提供的移动端平台`一次`又只能登录`一个账号`,每次切换账号又没有快捷方式;
>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;本项目旨在提供更加方便的管理方式,将会提供以下服务:
>
- 多账号管理
- 应用状态查询
- 自动隐藏`等待提交`状态的应用程序
- 查看反馈信息
- 回复反馈信息(无附件回复)
- 查看应用详情
- 修改部分应用详情处内容
- 查看应用程序的关键词覆盖
- 查看ASO榜单数据
